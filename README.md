# start

开始前安装eeui环境 (https://eeui.app/)

```
 npm install
 npm run dev
```

# ios
```
$ cd platforms/ios/eeuiApp
$ pod update                // 下载 iOS 依赖
$ open eeuiApp.xcworkspace // 自动打开项目
```
